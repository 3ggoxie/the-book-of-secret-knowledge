# Shell 插件


- [z](https://github.com/rupa/z) - 记录你最常用的文件夹，让你无需输入完整路径即可快速跳转。
- [fzf](https://github.com/junegunn/fzf) - 一个通用的命令行模糊查找工具。
- [zsh-autosuggestions](https://github.com/zsh-users/zsh-autosuggestions) - Fish 风格的 Zsh 自动补全建议。
- [zsh-syntax-highlighting](https://github.com/zsh-users/zsh-syntax-highlighting) - Fish 风格的 Zsh 语法高亮。
- [Awesome ZSH Plugins](https://github.com/unixorn/awesome-zsh-plugins) - ZSH 框架、插件、主题和教程列表。
