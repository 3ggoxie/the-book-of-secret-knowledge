# 网络 (HTTP)


- [curl](https://curl.haxx.se/) - 一个命令行工具和库，用于通过 URL 传输数据。
- [kurly](https://gitlab.com/davidjpeacock/kurly) - Golang 编写的广泛流行的 curl 程序的替代品。
- [HTTPie](https://github.com/jakubroztocil/httpie) - 一个用户友好的 HTTP 客户端。
- [wuzz](https://github.com/asciimoo/wuzz) - 用于 HTTP 检查的交互式 CLI 工具。
- [h2spec](https://github.com/summerwind/h2spec) - HTTP/2 实现的一致性测试工具。
- [h2t](https://github.com/gildasio/h2t) - 一个简单的工具，帮助系统管理员强化他们的网站。
- [htrace.sh](https://github.com/trimstray/htrace.sh) - 一个简单的 HTTP/HTTPS 故障排除和分析的瑞士军刀。
- [httpstat](https://github.com/reorx/httpstat) - 一个以美观清晰的方式可视化 curl 统计数据的工具。
- [httplab](https://github.com/gchaincl/httplab) - 一个交互式 Web 服务器。
- [Lynx](https://lynx.browser.org/) - 万维网的文本浏览器。
- [Browsh](https://github.com/browsh-org/browsh/) - 一个完全交互式、实时、现代的基于文本的浏览器。
- [HeadlessBrowsers](https://github.com/dhamaniasad/HeadlessBrowsers) - 现有（几乎）所有无头 Web 浏览器的列表。
- [ab](https://httpd.apache.org/docs/2.4/programs/ab.html) - 一个单线程命令行工具，用于测量 HTTP Web 服务器的性能。
- [siege](https://www.joedog.org/siege-home/) - HTTP 负载测试和基准测试实用程序。
- [wrk](https://github.com/wg/wrk) - 现代 HTTP 基准测试工具，能够产生显著的负载。
- [wrk2](https://github.com/giltene/wrk2) - wrk 的恒定吞吐量、正确延迟记录变体。
- [vegeta](https://github.com/tsenart/vegeta) - 恒定吞吐量、正确延迟记录变体。
- [bombardier](https://github.com/codesenberg/bombardier) - 用 Go 编写的快速跨平台 HTTP 基准测试工具。
- [gobench](https://github.com/cmpxchg16/gobench) - HTTP/HTTPS 负载测试和基准测试工具。
- [hey](https://github.com/rakyll/hey) - HTTP 负载生成器，ApacheBench (ab) 替代品，前身为 rakyll/boom。
- [boom](https://github.com/tarekziade/boom) - 一个脚本，可用于快速测试你的 Web 应用部署。
- [SlowHTTPTest](https://github.com/shekyan/slowhttptest) - 通过延长 HTTP 连接来模拟某些应用层拒绝服务攻击的工具。
- [gobuster](https://github.com/OJ/gobuster) - 一个自由开源的目录/文件 & DNS 爆破工具，用 Go 编写。
- [ssllabs-scan](https://github.com/ssllabs/ssllabs-scan) - SSL Labs API 的命令行参考实现客户端。
- [http-observatory](https://github.com/mozilla/http-observatory) - Mozilla HTTP Observatory CLI 版本。
- [Hurl](https://hurl.dev) - 一个用纯文本运行和测试 HTTP 请求的命令行工具。
