# 管理器


- [Midnight Commander](https://midnight-commander.org/) - 一个可视化文件管理器，基于 GNU General Public License 发布。
- [ranger](https://github.com/ranger/ranger) - 一个受 Vim 启发设计的 console 文件管理器。
- [nnn](https://github.com/jarun/nnn) - 一个小巧、快速、功能丰富的文件管理器。
- [screen](https://www.gnu.org/software/screen/) - 一个全屏窗口管理器，可在物理终端上实现多路复用。
- [tmux](https://github.com/tmux/tmux/wiki) - 一个终端复用器，让你能在一个终端中轻松切换多个程序。
- [tmux-cssh](https://github.com/peikk0/tmux-cssh) - 一个用于设置舒适易用的 tmux 会话的工具。
