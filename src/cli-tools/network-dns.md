# 网络 (DNS)


- [dnsdiag](https://github.com/farrokhi/dnsdiag) - DNS 诊断和性能测量工具。
- [fierce](https://github.com/mschwager/fierce) - 用于定位非连续 IP 空间的 DNS 侦察工具。
- [subfinder](https://github.com/subfinder/subfinder) - 发现网站有效子域名的子域名发现工具。
- [sublist3r](https://github.com/aboul3la/Sublist3r) - 面向渗透测试人员的快速子域名枚举工具。
- [amass](https://github.com/OWASP/Amass) - 通过爬取数据源、爬取 Web 档案等方式获取子域名的工具。
- [namebench](https://github.com/google/namebench) - 根据你的浏览历史提供个性化的 DNS 服务器推荐。
- [massdns](https://github.com/blechschmidt/massdns) - 高性能 DNS stub 解析器，用于批量查询和侦察。
- [knock](https://github.com/guelfoweb/knock) - 通过字典对目标域名的子域名进行枚举的工具。
- [dnsperf](https://github.com/DNS-OARC/dnsperf) - DNS 性能测试工具。
- [dnscrypt-proxy 2](https://github.com/jedisct1/dnscrypt-proxy) - 灵活的 DNS 代理，支持加密的 DNS 协议。
- [dnsdbq](https://github.com/dnsdb/dnsdbq) - 提供对被动 DNS 数据库系统访问的 API 客户端。
- [grimd](https://github.com/looterz/grimd) - 快速 dns 代理，用于屏蔽互联网广告和恶意服务器。
  - [dnstwist](https://github.com/elceef/dnstwist) - 检测 typosquatters、网络钓鱼攻击、欺诈和品牌仿冒。
