# Shells


- [GNU Bash](https://www.gnu.org/software/bash/) - 一个兼容 sh 的 Shell，集成了 Korn Shell 和 C Shell 的实用功能。
- [Zsh](https://www.zsh.org/) - 一个为交互使用而设计的 Shell，同时也支持强大的脚本语言。
- [tclsh](https://tcl-lang.org/) - 一个功能强大的跨平台 Shell，适用于多种用途。
- [bash-it](https://github.com/Bash-it/bash-it) - 一个用于开发、维护 Shell 脚本和自定义命令的框架。
- [Oh My ZSH!](https://ohmyz.sh/) - 最棒的 Zsh 配置管理框架。
- [Oh My Fish](https://github.com/oh-my-fish/oh-my-fish) - Fishshell 框架。
- [Starship](https://github.com/starship/starship) - 用 Rust 编写的跨平台提示符。
- [powerlevel10k](https://github.com/romkatv/powerlevel10k) - Powerlevel9k ZSH 主题的快速重实现。
