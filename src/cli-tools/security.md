# 安全


- [SELinux](https://access.redhat.com/documentation/en-us/red_hat_enterprise_linux/5/html/deployment_guide/ch-selinux) - 提供内置于 Linux 内核的灵活强制访问控制（MAC）系统。
- [AppArmor](https://wiki.ubuntu.com/AppArmor) - 主动保护操作系统和应用程序免受外部或内部威胁。
- [grapheneX](https://github.com/grapheneX/grapheneX) - 自动化系统强化框架。
- [DevSec Hardening Framework](https://github.com/dev-sec/) - 安全 + DevOps：自动服务器强化。
