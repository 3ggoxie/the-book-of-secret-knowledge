# 审计工具


- [ossec](https://www.ossec.net/) - 通过文件完整性监控主动监控系统活动的各个方面。
- [auditd](https://access.redhat.com/documentation/en-us/red_hat_enterprise_linux/6/html/security_guide/chap-system_auditing) - 提供在你的系统上跟踪安全相关信息的方法。
- [Tiger](https://www.nongnu.org/tiger/) - 一个可用作安全审计和入侵检测系统的安全工具。
- [Lynis](https://cisofy.com/lynis/) - 为运行 Linux、macOS 或基于 Unix 的操作系统的系统经过实战测试的安全工具。
- [LinEnum](https://github.com/rebootuser/LinEnum) - 用于本地 Linux 枚举和权限提升检查的脚本化工具。
- [Rkhunter](https://github.com/installation/rkhunter) - 用于 Linux 系统的扫描工具，用于扫描系统上的后门、rootkits 和本地漏洞利用。
- [PE-sieve](https://github.com/hasherezade/pe-sieve) - 一个轻量级工具，帮助检测在系统上运行的恶意软件。
- [PEASS](https://github.com/carlospolop/privilege-escalation-awesome-scripts-suite) - 用于 Windows 和 Linux/Unix 及 MacOS 的权限提升工具。
