# 文本编辑器


- [vi](http://ex-vi.sourceforge.net/) - Unix 系统上最常见的文本编辑器之一。
- [vim](https://www.vim.org/) - 一个高度可配置的文本编辑器。
- [emacs](https://www.gnu.org/software/emacs/) - 一个可扩展、可定制、自由开源的文本编辑器等功能。
- [micro](https://github.com/zyedidia/micro) - 一个现代、直观的终端文本编辑器。
- [neovim](https://neovim.io/) - 一个免费开源、功能强大、可扩展且易用的代码编辑器。
- [spacemacs](https://www.spacemacs.org/) - 一个社区驱动的 Emacs 发行版。
- [spacevim](https://spacevim.org/) - 一个社区驱动的 Vim 发行版。
