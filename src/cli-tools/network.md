# 网络


- [PuTTY](https://www.putty.org/) - 一个 SSH 和 telnet 客户端，最初由 Simon Tatham 开发。
- [Mosh](https://mosh.org/) - 一个 SSH 包装器，用于在不稳定连接下保持 SSH 会话活跃。
- [Eternal Terminal](https://eternalterminal.dev/) - 支持在 SSH 会话中使用鼠标滚动和 tmux 命令。
- [nmap](https://nmap.org/) - 一个用于网络发现和安全审计的自由开源（许可证）工具。
- [zmap](https://github.com/zmap/zmap) - 一个快速单包网络扫描器，专为 Internet 范围的网络调查设计。
- [Rust Scan](https://github.com/RustScan/RustScan) - 比 Nmap 更快地找到所有开放端口。
- [masscan](https://github.com/robertdavidgraham/masscan) - 最快的 Internet 端口扫描器，异步发送 SYN 数据包。
- [pbscan](https://github.com/gvb84/pbscan) - 一个更快、更高效的无状态 SYN 扫描器和 banner 获取工具。
- [hping](http://www.hping.org/) - 一个命令行-oriented 的 TCP/IP 数据包汇编器/分析器。
- [mtr](https://github.com/traviscross/mtr) - 一个将 traceroute 和 ping 程序功能结合在一个工具中的工具。
- [mylg](https://github.com/mehrdadrad/mylg) - 将不同网络探针的功能组合在一个诊断工具中的实用程序。
- [netcat](http://netcat.sourceforge.net/) - 使用 TCP/IP 协议跨网络连接读取和写入数据的实用程序。
- [socat](http://www.dest-unreach.org/socat/) - 在两个对象之间传输数据的实用程序。
- [tcpdump](https://www.tcpdump.org/) - 一个强大的命令行数据包分析器。
- [tshark](https://www.wireshark.org/docs/man-pages/tshark.html) - 一个允许我们转储和分析网络流量（Wireshark CLI）的工具。
- [Termshark](https://termshark.io/) - 一个简单的 tshark 终端用户界面。
- [ngrep](https://github.com/jpr5/ngrep) - 就像 GNU grep 应用于网络层。
- [netsniff-ng](http://netsniff-ng.org/) - 你的日常 Linux 网络管理的瑞士军刀。
- [sockdump](https://github.com/mechpen/sockdump) - 转储 unix 域套接字流量。
- [stenographer](https://github.com/google/stenographer) - 一个数据包捕获解决方案，旨在快速将所有数据包写入磁盘。
- [tcpterm](https://github.com/sachaos/tcpterm) - 在 TUI 中可视化数据包。
- [bmon](https://github.com/tgraf/bmon) - 一个监控和调试工具，用于捕获网络相关统计信息并以可视化方式呈现。
- [iptraf-ng](http://iptraf.seul.org/2.6/manual.html#installation) - 一个面向 Linux 的基于控制台的网络监控程序，显示关于 IP 流量信息。
- [vnstat](https://github.com/vergoh/vnstat) - Linux 和 BSD 的网络流量监控器。
- [iPerf3](https://iperf.fr/) - 用于主动测量 IP 网络上最大可达带宽的工具。
- [ethr](https://github.com/Microsoft/Ethr) - TCP、UDP 和 HTTP 的网络性能测量工具。
- [Etherate](https://github.com/jwbensley/Etherate) - 基于 Linux CLI 的 Ethernet 和 MPLS 流量测试工具。
- [echoip](https://github.com/mpolden/echoip) - IP 地址查询服务。
- [Nemesis](https://github.com/troglobit/nemesis) - 数据包操作 CLI 工具；制作和注入多种协议的 packets。
- [packetfu](https://github.com/packetfu/packetfu) - 用于 Ruby 的中级数据包操作库。
- [Scapy](https://scapy.net/) - 数据包操作库；伪造、发送、解码、捕获多种协议的 packets。
- [impacket](https://github.com/SecureAuthCorp/impacket) - 用于处理网络协议的 Python 类集合。
- [ssh-audit](https://github.com/arthepsy/ssh-audit) - SSH 服务器审计工具。
- [aria2](https://aria2.github.io/) - 一个轻量级多协议多源命令行下载实用程序。
- [iptables-tracer](https://github.com/x-way/iptables-tracer) - 观察数据包通过 iptables 链的路径。
- [inception](https://github.com/proabiral/inception) - 一个高度可配置的工具，可以根据任意数量的主机检查任意内容。
- [mRemoteNG](https://mremoteng.org/) - mRemote 的一个分支，多标签页 PuTTy 增强版！
