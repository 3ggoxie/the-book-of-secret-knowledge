# 终端模拟器


- [Guake](https://github.com/Guake/guake) - 为 GNOME 桌面环境设计的下拉式终端。
- [Terminator](https://gnometerminator.blogspot.com/p/introduction.html) - 基于 GNOME Terminal，为系统管理员和其他用户提供有用的功能。
- [Kitty](https://sw.kovidgoyal.net/kitty/) - 基于 GPU 的终端模拟器，支持平滑滚动和图片显示。
- [Alacritty](https://github.com/alacritty/alacritty) - 一个快速、跨平台的 OpenGL 终端模拟器。
