# Web 训练应用（本地安装）


- [OWASP-VWAD](https://www.owasp.org/index.php/OWASP_Vulnerable_Web_Applications_Directory_Project) - 所有已知漏洞 Web 应用程序的全面且维护良好的注册表。
- [DVWA](http://www.dvwa.co.uk/) - PHP/MySQL Web 应用程序，非常漏洞。
- [metasploitable2](https://metasploit.help.rapid7.com/docs/metasploitable-2) - 在安全研究人员中流行的漏洞 Web 应用程序。
- [metasploitable3](https://github.com/rapid7/metasploitable3) - 从头构建的 VM，具有大量安全漏洞。
- [DSVW](https://github.com/stamparm/DSVW) - 是一个故意漏洞的 Web 应用程序，用不到 100 行代码编写。
- [OWASP Mutillidae II](https://sourceforge.net/projects/mutillidae/) - 免费的、开源的、故意漏洞的 Web 应用程序。
- [OWASP Juice Shop Project](https://www.owasp.org/index.php/OWASP_Juice_Shop_Project) - 存在的最无漏洞的漏洞应用程序。
- [OWASP Node js Goat Project](https://www.owasp.org/index.php/Projects/OWASP_Node_js_Goat_Project) - OWASP Top 10 安全风险适用于使用 Node.js 开发的 Web 应用程序。
- [juicy-ctf](https://github.com/iteratec/juicy-ctf) - 使用 OWASP Juice Shop 运行夺旗和安全培训。
- [SecurityShepherd](https://github.com/OWASP/SecurityShepherd) - Web 和移动应用程序安全培训平台。
- [Security Ninjas](https://github.com/opendns/Security_Ninjas_AppSec_Training) - 开源应用程序安全培训程序。
- [hackazon](https://github.com/rapid7/hackazon) - 一个现代漏洞 Web 应用程序。
- [dvna](https://github.com/appsecco/dvna) - 非常漏洞的 NodeJS 应用程序。
- [django-DefectDojo](https://github.com/DefectDojo/django-DefectDojo) - 一个开源应用程序漏洞关联和安全编排工具。
- [Google Gruyere](https://google-gruyere.appspot.com/) - Web 应用程序漏洞利用和防御。
- [Bodhi](https://github.com/amolnaik4/bodhi) - 是一个专注于学习客户端 Web 漏洞利用的操场。
- [Websploit](https://websploit.h4cker.org/) - 单个 VM 实验室，旨在将多个漏洞应用程序组合在一个环境中。
- [vulhub](https://github.com/vulhub/vulhub) - 基于 docker-compose 的预构建漏洞环境。
- [CloudGoat 2](https://rhinosecuritylabs.com/aws/introducing-cloudgoat-2/) - 新的和改进的"按设计漏洞"
AWS 部署工具。
- [secDevLabs](https://github.com/globocom/secDevLabs) - 是一个以实用方式学习安全 Web 开发的实验室。
- [CORS-vulnerable-Lab](https://github.com/incredibleindishell/CORS-vulnerable-Lab) - 示例漏洞代码及其利用代码。
- [RootTheBox](https://github.com/moloch--/RootTheBox) - 黑客游戏（CTF 计分板和游戏管理器）。
- [KONTRA](https://application.security/) - 应用程序安全培训（OWASP Top Web & Api）。
