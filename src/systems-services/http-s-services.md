# HTTP(s) 服务


- [Varnish Cache](https://varnish-cache.org/) - 为内容繁重的动态网站设计的 HTTP 加速器。
- [Nginx](https://nginx.org/) - 开源 Web 和反向代理服务器，类似于 Apache，但非常轻量级。
- [OpenResty](https://openresty.org/en/) - 基于 NGINX 和 LuaJIT 的动态 Web 平台。
- [Tengine](https://github.com/alibaba/tengine) - 具有一些高级功能的 Nginx 发行版。
- [Caddy Server](https://caddyserver.com/) - 一个开源的、默认启用 HTTP/2 的 Web 服务器，默认启用 HTTPS。
- [HAProxy](https://www.haproxy.org/) - 可靠、高性能的 TCP/HTTP 负载均衡器。
