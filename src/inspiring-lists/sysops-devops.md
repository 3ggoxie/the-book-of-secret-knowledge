# SysOps/DevOps


- [Awesome Sysadmin](https://github.com/kahun/awesome-sysadmin) - 令人惊叹的开源系统管理员资源。
- [Awesome Shell](https://github.com/alebcay/awesome-shell) - 令人惊叹的命令行框架、工具包、指南和小工具。
- [Command-line-text-processing](https://github.com/learnbyexample/Command-line-text-processing) - 查找要搜索和替换的文本，排序以美化，以及更多。
- [Awesome Pcaptools](https://github.com/caesar0301/awesome-pcaptools) - 由其他研究人员开发的处理网络跟踪的工具集合。
- [awesome-ebpf](https://github.com/zoidbergwill/awesome-ebpf) - 与 eBPF 相关的令人惊叹的项目列表。
- [Linux Network Performance](https://github.com/leandromoreira/linux-network-performance-parameters) - 一些网络 sysctl 变量如何适应 Linux/Kernel 网络流。
- [Awesome Postgres](https://github.com/dhamaniasad/awesome-postgres) - 令人惊叹的 PostgreSQL 软件、库、工具和资源列表。
- [quick-SQL-cheatsheet](https://github.com/enochtangg/quick-SQL-cheatsheet) - 所有 SQL 查询及其使用示例的快速提醒。
- [Awesome-Selfhosted](https://github.com/Kickball/awesome-selfhosted) - 可在本地托管的免费软件网络服务和 Web 应用程序列表。
- [List of applications](https://wiki.archlinux.org/index.php/List_of_applications) - 按类别排序的大量应用程序列表，作为寻找包的人的参考。
- [CS-Interview-Knowledge-Map](https://github.com/InterviewMap/CS-Interview-Knowledge-Map) - 建立最好的面试地图。
- [DevOps-Guide](https://github.com/Tikam02/DevOps-Guide) - 从基础到高级的 DevOps 指南，包含面试问题和笔记。
- [FreeBSD Journal](https://issue.freebsdfoundation.org/publication/?m=33057&l=1&view=issuelistBrowser) - 关于 FreeBSD 和其他重要事物的定期杂志的伟大列表。
- [devops-interview-questions](https://github.com/bregman-arie/devops-interview-questions) - 包含各种 DevOps 和 SRE 相关主题的面试问题。
