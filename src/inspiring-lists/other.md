# 其他


- [Cheatography](https://www.cheatography.com/) - 超过 3000 张免费速查表、复习辅助工具和快速参考。
- [awesome-static-analysis](https://github.com/mre/awesome-static-analysis) - 适用于所有编程语言的静态分析工具。
- [computer-science](https://github.com/ossu/computer-science) - 在计算机科学中免费自学教育的路径。
- [post-mortems](https://github.com/danluu/post-mortems) - 事后分析集合（配置错误、硬件故障等）。
- [build-your-own-x](https://github.com/danistefanovic/build-your-own-x) - 构建你自己的（在此处插入技术）。
- [Project-Based-Tutorials-in-C](https://github.com/rby90/Project-Based-Tutorials-in-C) - C 中基于项目的教程精选列表。
- [The-Documentation-Compendium](https://github.com/kylelobo/The-Documentation-Compendium) - 各种 README 模板和关于编写高质量文档的提示。
- [awesome-python-applications](https://github.com/mahmoud/awesome-python-applications) - 免费软件，运行良好，而且恰好是开源 Python。
- [awesome-public-datasets](https://github.com/awesomedata/awesome-public-datasets) - 以主题为中心的高质量开放数据集列表。
- [machine-learning-algorithms](https://github.com/Sahith02/machine-learning-algorithms) - 所有机器学习算法和概念的精选列表。
