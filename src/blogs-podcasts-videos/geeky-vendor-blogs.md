- [Tripwire State of Security](https://www.tripwire.com/state-of-security/) - 博客，提供关于当前安全问题的最新新闻、趋势和见解。

- [TrustedSec](https://www.trustedsec.com/category/articles/) - 关于网络安全的最新新闻和趋势。
- [PortSwigger Web Security Blog](https://portswigger.net/blog) - 关于 Web 应用安全漏洞以及网络安全团队的最佳建议。
- [AT&T Cybersecurity blog](https://www.alienvault.com/blogs) - 关于新兴威胁的新闻和简化威胁检测的实用建议。
