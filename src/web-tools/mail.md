# 邮件


- [smtp-tls-checker](https://luxsci.com/smtp-tls-checker) - 检查电子邮件域的 SMTP TLS 支持。
- [MX Toolbox](https://mxtoolbox.com/SuperTool.aspx) - 你的所有 MX 记录、DNS、黑名单和 SMTP 诊断都在一个综合工具中。
- [Secure Email](https://www.checktls.com/index.html) - 面向电子邮件技术人员的完整邮件测试工具。
- [blacklistalert](http://www.blacklistalert.org/) - 检查你的域名是否在实时垃圾邮件黑名单上。
- [MultiRBL](http://multirbl.valli.org/) - 发送邮件服务器的完整 IP 检查。
- [DKIM SPF & Spam Assassin Validator](https://dkimvalidator.com/) - 检查邮件认证并用 Spam Assassin 对邮件评分。
