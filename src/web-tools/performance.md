# 性能


- [GTmetrix](https://gtmetrix.com/) - 分析你网站的速度并让它更快。
- [Sucuri loadtimetester](https://performance.sucuri.net/) - 从全球各地测试你网站的性能。
- [Pingdom Tools](https://tools.pingdom.com/) - 在全球范围内分析你网站的速度。
- [PingMe.io](https://pingme.io/) - 在多个地理区域运行网站延迟测试。
- [PageSpeed Insights](https://developers.google.com/speed/pagespeed/insights/) - 分析你网站的速度并让它更快。
- [web.dev](https://web.dev/) - 帮助像你一样的开发者学习和应用 Web 的现代功能到你自己的网站和应用程序。
- [Lighthouse](https://github.com/GoogleChrome/lighthouse) - 面向 Web 的自动化审计、性能指标和最佳实践。
