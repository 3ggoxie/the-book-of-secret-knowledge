# 大规模扫描器（搜索引擎）


- [Censys](https://censys.io/) - 帮助信息安全从业者发现、监控和分析设备的平台。
- [Shodan](https://www.shodan.io/) - 首个面向互联网连接设备的搜索引擎。
- [Shodan 2000](https://2000.shodan.io/#/) - 这个工具查找 Shodan 随机生成的数据。
- [GreyNoise](https://viz.greynoise.io/table) - 与 Shodan 和 Censys 类似的大规模扫描器。
- [ZoomEye](https://www.zoomeye.org/) - 网络空间搜索引擎，让用户找到特定的网络组件。
- [netograph](https://netograph.io/) - 监控和理解 Web 深层结构的工具。
- [FOFA](https://fofa.so/) - 一个网络空间搜索引擎。
- [onyphe](https://www.onyphe.io/) - 用于收集开源和网络威胁情报数据的搜索引擎。
- [IntelligenceX](https://intelx.io/) - 一个搜索引擎和数据档案馆。
- [binaryedge](https://app.binaryedge.io/) - 扫描整个互联网空间并创建实时威胁情报流和报告。
- [Spyse](https://spyse.com/) - 互联网资产注册表：网络、威胁、Web 对象等。
- [wigle](https://wigle.net/) - 基于提交无线网络目录。所有的网络。被所有人发现。
- [PublicWWW](https://publicwww.com/) - 在网页 HTML、JS 和 CSS 代码中查找任何字母数字片段、签名或关键字。
- [IntelTechniques](https://inteltechniques.com/index.html) - 这个仓库包含数百个在线搜索工具。
- [hunter](https://hunter.io/) - 让你在几秒钟内找到电子邮件地址，并与对你的业务重要的人联系。
- [GhostProject?](https://ghostproject.fr/) - 按完整电子邮件地址或用户名搜索。
- [databreaches](https://www.databreaches.live/) - 我的电子邮件是否受到数据泄露影响？
- [We Leak Info](https://weleakinfo.com) - 世界上最快和最大的数据泄露搜索引擎。
- [Pulsedive](https://pulsedive.com/) - 扫描恶意 URL、IP 和域名，包括端口扫描和 Web 请求。
- [Buckets by Grayhatwarfar](https://buckets.grayhatwarfare.com/) - 公共搜索 Open Amazon S3 Buckets 及其内容的数据库。
- [Vigilante.pw](https://vigilante.pw/) - 泄露数据库目录。
- [builtwith](https://builtwith.com/) - 找出网站是用什么构建的。
- [NerdyData](https://nerdydata.com/) - 在 Web 源代码中搜索技术，跨越数百万个站点。
- [zorexeye](http://zorexeye.com/) - 搜索网站、图片、应用程序、软件等。
- [Mamont's open FTP Index](https://www.mmnt.net/) - 如果目标有一个开放的 FTP 站点与可访问的内容，它将列在这里。
- [OSINT Framework](https://osintframework.com/) - 专注于从免费工具或资源收集信息。
- [maltiverse](https://www.maltiverse.com/search) - 面向网络安全分析师的服务。
- [Leaked Source](https://leakedsource.ru/main/) - 以查找形式出现的在线数据协作。
- [We Leak Info](https://search.weleakinfo.com/) - 帮助日常个人保护他们的在线生活，避免被黑客攻击。
- [pipl](https://pipl.com/) - 查找电子邮件地址、社交用户名或电话号码背后的人的地方。
- [abuse.ch](https://abuse.ch/) - 由一个随机的瑞士人运营的非营利性恶意软件斗争。
- [malc0de](http://malc0de.com/database/) - 恶意软件搜索引擎。
- [Cybercrime Tracker](https://cybercrime-tracker.net/index.php) - 监控和追踪用于实施网络犯罪的各种恶意软件家族。
- [shhgit](https://github.com/eth0izzle/shhgit/) - 实时查找 GitHub 密钥。
- [searchcode](https://searchcode.com/) - 帮助你找到函数、API 和库的真实示例。
- [Insecam](http://www.insecam.org/) - 世界上最大的在线监控安全摄像头目录。
- [index-of](http://index-of.es/) - 包含很多好东西，如：安全、黑客、逆向工程、密码学、编程等。
- [Rapid7 Labs Open Data](https://opendata.rapid7.com/) - 来自 Project Sonar 的数据集的宝贵资源。
- [Common Response Headers](https://webtechsurvey.com/common-response-headers) - 最大的 HTTP 响应头数据库。
- [InQuest Labs](https://labs.inquest.net) - InQuest Labs 是一个面向安全研究人员的开放、交互式和 API 驱动的数据门户。
