# 编码器/解码器和 Regex 测试


- [URL Encode/Decode](https://www.url-encode-decode.com/) - 用于编码或解码文本字符串的工具。
- [Uncoder](https://uncoder.io/) - 日志数据搜索查询的在线翻译器。
- [Regex101](https://regex101.com/) - 在线 regex 测试器和调试器：PHP、PCRE、Python、Golang 和 JavaScript。
- [RegExr](https://regexr.com/) - 用于学习、构建和测试正则表达式的在线工具。
- [RegEx Testing](https://www.regextester.com/) - 在线 regex 测试工具。
- [RegEx Pal](https://www.regexpal.com/) - 在线 regex 测试工具 + 其他工具。
- [The Cyber Swiss Army Knife](https://gchq.github.io/CyberChef/) - 一个用于加密、编码、压缩和数据分析的 Web 应用程序。
