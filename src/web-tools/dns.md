# DNS


- [ViewDNS](http://viewdns.info/) - 免费 DNS 相关工具和信息的单一来源。
- [DNSLookup](https://dnslookup.org/) - 一个高级 DNS 查询工具。
- [DNSlytics](https://dnslytics.com/) - 在线 DNS 调查工具。
- [DNS Spy](https://dnsspy.io/) - 监控、验证你的 DNS 配置。
- [Zonemaster](https://zonemaster.iis.se/en/) - 帮助你控制 DNS 的运行方式。
- [Leaf DNS](http://leafdns.com/) - 全面的 DNS 测试器。
- [Find subdomains online](https://findsubdomains.com/) - 为安全评估渗透测试查找子域名。
- [DNSdumpster](https://dnsdumpster.com/) - DNS 侦察和研究，查找 DNS 记录。
- [DNS Table online](https://dnstable.com/) - 按域名、IP、CIDR、ISP 搜索 DNS 记录。
- [intoDNS](https://intodns.com/) - DNS 和邮件服务器健康检查器。
- [DNS Bajaj](http://www.zonecut.net/dns/) - 检查你域名的委派。
- [BuddyDNS Delegation LAB](https://www.buddyns.com/delegation-lab/) - 检查、追踪和可视化你域名的委派。
- [dnssec-debugger](https://dnssec-debugger.verisignlabs.com/) - DS 或 DNSKEY 记录验证器。
- [PTRarchive.com](http://ptrarchive.com/) - 这个站点负责保存历史反向 DNS 记录。
- [xip.io](http://xip.io/) - 面向所有人的通配符 DNS。
- [nip.io](https://nip.io/) - 任意 IP 地址的死简单通配符 DNS。
- [dnslookup (ceipam)](https://ceipam.eu/en/dnslookup.php) - 最好的 DNS 传播检查器之一（不仅如此）。
- [What's My DNS](https://whatsmydns.com) - DNS 传播检查工具。
- [DNSGrep](https://blog.erbbysam.com/index.php/2019/02/09/dnsgrep/) - 快速搜索大型 DNS 数据集。
