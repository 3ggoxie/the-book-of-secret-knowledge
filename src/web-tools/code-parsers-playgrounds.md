# 代码解析器/playgrounds


- [ShellCheck](https://www.shellcheck.net/) - 查找你的 shell 脚本中的 bug。
- [explainshell](https://explainshell.com/) - 获取 shell 命令的交互式帮助文本。
- [jsbin](https://jsbin.com/?html,output) - HTML、CSS & JavaScript 的实时 pastebin 等。
- [CodeSandbox](https://codesandbox.io/) - 用于 Web 应用程序开发的在线代码编辑器。
- [PHP Sandbox](http://sandbox.onlinephpfunctions.com/) - 用这个代码测试器测试你的 PHP 代码。
- [Repl.it](https://www.repl.it/) - 一个即时 IDE，集学习、构建、协作和托管于一体。
- [vclFiddle](http://www.vclfiddle.net/) - 用于试验 Varnish Cache VCL 的在线工具。
- [Haskell Dockerfile Linter](https://github.com/hadolint/hadolint) - 一个更智能的 Dockerfile linter，帮助你构建最佳实践 Docker 镜像。
