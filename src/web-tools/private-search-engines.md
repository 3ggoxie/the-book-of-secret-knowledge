# 私密搜索引擎


- [Startpage](https://www.startpage.com/) - 世界上最私密的搜索引擎。
- [searX](https://searx.me/) - 尊重隐私、可 hack 的元搜索引擎。
- [darksearch](https://darksearch.io/) - 第一个真正的暗网搜索引擎。
- [Qwant](https://www.qwant.com/) - 尊重你隐私的搜索引擎。
- [DuckDuckGo](https://duckduckgo.com/) - 不会追踪你的搜索引擎。
- [Swisscows](https://swisscows.com/) - 安全的网络搜索。
- [Disconnect](https://search.disconnect.me/) - 使你的搜索匿名化的搜索引擎。
- [MetaGer](https://metager.org/) - 使用匿名代理和隐藏 Tor 分支的搜索引擎。
