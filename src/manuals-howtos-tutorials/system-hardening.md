# 系统强化


- [CIS Benchmarks](https://www.cisecurity.org/cis-benchmarks/) - 超过 100 个技术的安全配置设置，可作为免费 PDF。
- [Security Harden CentOS 7](https://highon.coffee/blog/security-harden-centos-7/) - 这将引导你完成保护 CentOS 的步骤。
- [CentOS 7 Server Hardening Guide](https://www.lisenet.com/2017/centos-7-server-hardening-guide/) - 保护 CentOS 的出色指南；熟悉 OpenSCAP。
- [awesome-security-hardening](https://github.com/decalage2/awesome-security-hardening) - 安全强化指南、工具和其他资源的集合。
- [The Practical Linux Hardening Guide](https://github.com/trimstray/the-practical-linux-hardening-guide) - 提供强化 GNU/Linux 系统的高级概述。
- [Linux Hardening Guide](https://madaidans-insecurities.github.io/guides/linux-hardening.html) - 如何尽可能地强化 Linux 以提高安全性和隐私性。
