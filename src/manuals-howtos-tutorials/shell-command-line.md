# Shell/命令行


- [pure-bash-bible](https://github.com/dylanaraps/pure-bash-bible) - 纯 bash 替代外部进程的集合。
- [pure-sh-bible](https://github.com/dylanaraps/pure-sh-bible) - 纯 POSIX sh 替代外部进程的集合。
- [bash-guide](https://github.com/Idnan/bash-guide) - 学习 bash 的指南。
- [bash-handbook](https://github.com/denysdovhan/bash-handbook) - 面向想要学习 Bash 的人。
- [The Bash Hackers Wiki](https://wiki.bash-hackers.org/start) - 保存 GNU Bash 各种文档。
- [Shell & Utilities](http://pubs.opengroup.org/onlinepubs/9699919799/utilities/contents.html) - 描述 POSIX 兼容系统向应用程序提供的命令。
- [the-art-of-command-line](https://github.com/jlevy/the-art-of-command-line) - 在一页纸上掌握命令行。
- [Shell Style Guide](https://google.github.io/styleguide/shellguide.html) - 面向 Google 发起的开源项目的 shell 风格指南。
