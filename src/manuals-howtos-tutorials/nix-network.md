# *nix & 网络


- [nixCraft](https://www.cyberciti.biz/) -面向新人和资深系统管理员的 Linux 和 Unix 教程。
- [TecMint](https://www.tecmint.com/) - 面向系统管理员和极客的理想 Linux 博客。
- [Omnisecu](http://www.omnisecu.com/index.php) - 免费的网络、系统管理和安全教程。
- [linux-cheat](https://github.com/cirosantilli/linux-cheat) - Linux 教程和速查表。最小示例。大多数是用户态 CLI 实用程序。
- [linuxupskillchallenge](https://github.com/snori74/linuxupskillchallenge) - 学习成为系统管理员所需的技能。
- [Unix Toolbox](http://cb.vu/unixtoolbox.xhtml) - 对 IT 工作或高级用户有用的 Unix/Linux/BSD 命令和任务。
- [Linux Kernel Teaching](https://linux-kernel-labs.github.io/refs/heads/master/index.html) - Linux 内核主题的讲座和实验室集合。
- [htop explained](https://peteris.rocks/blog/htop/) - 解释你在 Linux 上的 htop/top 中可以看到的一切。
- [Linux Guide and Hints](https://linuxguideandhints.com/) - 关于在 Fedora 和 CentOS 中系统管理的教程。
- [strace-little-book](https://github.com/NanXiao/strace-little-book) - 一本介绍 strace 的小书。
- [linux-tracing-workshop](https://github.com/goldshtn/linux-tracing-workshop) - Linux 追踪工具研讨会的示例和动手实验室。
- [http2-explained](https://github.com/bagder/http2-explained) - 解释和记录 HTTP/2 的详细文档。
- [http3-explained](https://github.com/bagder/http3-explained) - 描述 HTTP/3 和 QUIC 协议的文档。
- [HTTP/2 in Action](https://www.manning.com/books/http2-in-action) - 对新 HTTP/2 标准的出色介绍。
- [Let's code a TCP/IP stack](https://www.saminiir.com/lets-code-tcp-ip-stack-1-ethernet-arp/) - 很棒的东西，用于在更深入的层次上学习网络和系统编程。
- [Nginx Admin's Handbook](https://github.com/trimstray/nginx-admins-handbook) - 如何改善 NGINX 性能、安全和其他重要的事情。
- [nginxconfig.io](https://github.com/digitalocean/nginxconfig.io) - NGINX 配置生成器增强版。
- [openssh guideline](https://infosec.mozilla.org/guidelines/openssh) - 帮助运维团队配置 OpenSSH 服务器和客户端。
- [SSH Handshake Explained](https://gravitational.com/blog/ssh-handshake-explained/) - 对 SSH 握手的简要描述。
- [ISC's Knowledgebase](https://kb.isc.org/docs/using-this-knowledgebase) - 你可以找到关于 BIND 9、ISC DHCP 和 Kea DHCP 的一些一般信息。
- [PacketLife.net](https://packetlife.net/) - 在学习思科的 CCNP 认证时记录笔记的地方。
